Research

Currently, my research focuses on algebraic machine learning under the supervision of Bernd Sturmfels. I am particularly interested in high-dimensional data analysis, statistical methods for machine learning, and the application of these methods to complex systems.

Some specific areas of interest include:

  • Bayesian inference for inverse problems and partial differential equations (PDEs)
  • Generative models such as NeuralODEs
  • Algorithms for representative citizen selection in democratic decision-making

Projects

At Project Eaden, I am working as a data scientist, developing tools for analytics in the food science space. I created a dashboard using Python and Dash for analyzing experimental data from TPA and WB experiments. The dashboard allows for:

  • Data visualization through bar charts, line graphs, and interactive controls
  • Filtering and exporting capabilities for deep data analysis
  • Applying machine learning techniques for predictive modeling

My project integrates machine learning models to optimize the analysis process, leveraging AI to enhance decision-making in experimental food science. You can learn more about this work and other projects by exploring my GitHub.
